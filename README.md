# Brute Force Attack Prevention System

## Description
This project is a Python-based tool that simulates login attempts and implements basic protection mechanisms against brute force attacks.

## Features
- Limits multiple login attempts  
- Blocks access after repeated failures  
- Simulates real-world brute force attack scenarios  
- Displays warning or lockout messages  

## Technologies Used
- Python

## How to Run
1. Download the file  
2. Run the script:  
   python main.py  

## Example Output
Attempt 1: Incorrect password  
Attempt 2: Incorrect password  
Attempt 3: Access blocked due to multiple failed attempts
